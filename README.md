# ONIRIA Stone Town Brain

## Introduction
The ONIRIA Stone Town Brain is a connected knowledge and operating system for ONIRIA Stone Town, a 20-room luxury boutique hotel in Stone Town, Zanzibar. The Brain will help people and AI systems understand and operate every important part of the hotel.   

# Main Goal
The main goal is to create a connected, intelligent, and reliable Obsidian Brain that allows Lulu and the hotel team to operate ONIRIA Stone Town as a profitable, personalized, and high-quality five-star boutique hotel. 
## Design Thinking Methodology
- **Empathize:** Understand hotel users and Lulu, map the guest journey, and identify information needs.
- **Define and Ideate:** Design the system structure, note types, and dashboards.
- **Prototype Foundations:** Create folders, templates, dashboards, and sample notes.
- **Connect and Test:** Link notes, test key questions, identify missing connections, and improve the system.
