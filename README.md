# ONIRIA Stone Town Brain

A connected Obsidian knowledge and operating system for ONIRIA Stone Town, a 20-key AI-native luxury boutique hotel in Stone Town, Zanzibar.

## Open first

- `00 Control/ONIRIA Brain Dashboard.md`
- `01 Sources/Source Index.md`

## Completed scope in this package

- **Task 2:** Property, floors, room categories, Rooms 01-20, public areas, building systems and room inspection.
- **Task 3:** Connected priority SOPs, service standards and daily/opening/closing checklists.
- **Task 4:** Equipment, maintenance, inspections, warranties, suppliers, inventory, spares and reorder controls.
- **Task 5:** Departments, staff roles, onboarding, training, duty structure, safety, privacy and compliance.
