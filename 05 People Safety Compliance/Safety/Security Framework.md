# Security Framework

## Controls

- Role-based room and back-of-house access
- Credential issue, return, loss and audit
- Visitor/contractor control
- Key and master-access control
- Incident reporting and evidence preservation
- CCTV only where approved, lawful and proportionate
- Cash/payment security
- Guest identity and room-number privacy
- Missing person, threat and suspected-crime escalation
