# Supplier Profiles

## Purpose

Supplier profiles keep operational suppliers searchable by category, reliability, products, warranty responsibility, shipping ability, and service history.

## Supplier Profile Template

| Field | Description |
|---|---|
| Supplier name | Legal or trading name |
| Category | Equipment, maintenance, F&B, spa, linen, cleaning, IT, safety |
| Contact channel | Approved email or text contact |
| Products or services | Operationally useful items only |
| Ships to Zanzibar | Yes, no, unknown |
| Lead time | Confirmed or estimated |
| Certifications | Where relevant |
| Warranty role | Manufacturer, distributor, installer, service agent |
| Related equipment | Link to [[Equipment Inventory]] |
| Related inventory | Link to [[Inventory System]] |
| Performance record | Link to [[Supplier Performance Records]] |
| Status | Research, shortlisted, approved, active, suspended, retired |
| Source | Catalog, quote, contract, email, inspection |

## Supplier Communication Rule

Research catalog, specs, lead time, certifications, and Zanzibar shipping before contacting a supplier. Ask only for missing quote, availability, delivery, or unclear technical information.
