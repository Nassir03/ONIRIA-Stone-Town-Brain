# Equipment Inventory

## Purpose

This register stores operational equipment that must be maintained, repaired, inspected, replaced, covered by warranty, or linked to guest experience.

## Equipment Template

| Field | Description |
|---|---|
| Equipment ID | Unique asset ID |
| Name | Operational name |
| Category | HVAC, lock, lighting, KNX, pool, plumbing, F&B, spa, safety, IT |
| Brand / model | Exact product information |
| Serial number | If available |
| Room or area | Link to room or public area |
| Building system | Link to [[Building Systems Register]] |
| Supplier | Link to [[Supplier Profiles]] |
| Installer | Person or company |
| Warranty | Link to [[Warranty Records]] |
| Manual | Link to REFERENCE file |
| Maintenance schedule | Link to [[Preventive Maintenance Schedule]] |
| Guest impact | None, low, medium, high, critical |
| Safety impact | None, low, medium, high, critical |
| Status | Planned, ordered, installed, commissioned, active, repair, retired |
| REFERENCE | Approved document or inspection |

## Initial Equipment Categories

- Guest-room AC units
- Smart locks
- KNX devices
- Lighting fixtures and drivers
- Wi-Fi access points
- Speakers and audio equipment
- Occupancy or comfort sensors
- Bathroom fixtures requiring maintenance
- Private pool and terrace pool equipment
- Rooftop pool plant
- Spa equipment
- F&B refrigeration and bar equipment
- Fire-safety equipment
- Security equipment
