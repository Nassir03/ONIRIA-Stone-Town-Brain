# Inventory System

## Purpose

The inventory system tracks stock levels for operational supplies, spare parts, amenities, cleaning products, spa products, F&B items, and safety consumables.

## Inventory Fields

- Item ID
- Item name
- Category
- Unit
- Current quantity
- Minimum level
- Reorder level
- Target par
- Storage location
- Supplier
- Lead time
- Expiry date
- Safety or compliance requirement
- Last count date
- Counted by
- Status

## Status Values

- In stock
- Low stock
- Reorder now
- Ordered
- Backordered
- Expired
- Hold
- Retired
